# css-algorithm-python
Demonstration of the Content Scramble System (CSS) algorithm, utlized on commercial DVDs

Dependencies
===
Python 3  

Installation
===
Edit the plaintext and keyword variables in main.py, then run:
```
$ /path/to/main.py
```
